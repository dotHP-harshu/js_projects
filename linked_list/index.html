1.Type below function in input box for performing operations.

2.Then hit enter or click on run btn.

Functions:

  • iBeg(value) : insert given value at the beginning of the head.
  • iEnd(value) : insert given value at the end of the head.
  • iPos(position, value) : insert given value at the given position in the list.
  • rBeg() : remove the element at the beginning of list.
  • rEnd() : remove the element at the end of list.
  • rPos(position) : remove element by given position in the list.
  • rVal(value) : remove firs element by given value in the list.
  • rBeg() : remove the element from beginning in the list.
  • rEnd() : remove the element from end in the list.
  • traverse() : visit all elements in the list.
