Press ctrl+shift+i to open console tab.

Use console tab as code editor and run functions for operations.

Functions:

  • i_beg(value) : insert given value at the beginning of the head.
  • i_end(value) : insert given value at the end of the head.
  • i_pos(position, value) : insert given value at the given position in the list.
  • r_pos(position) : remove element by given position in the list.
  • r_val(value) : remove firs element by given value in the list.
